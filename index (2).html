Gemini API Key
請貼上 Gemini API key。Key 只會暫存在本分頁 sessionStorage，關閉分頁後即清除；貼上後會延遲自動測試一次，並加入冷卻機制避免 Too many requests。隨機召喚與本機詩庫查詢不會改變 API 連線狀態。未輸入 key 時仍可使用本機詩庫與免費瀏覽器朗讀。
